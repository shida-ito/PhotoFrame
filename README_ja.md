# PhotoFrame

macOS向けのスタイリッシュでミニマルな写真フレーム作成アプリ。写真や動画にエレガントなボーダーと、metadata をもとにしたカスタマイズ可能なテキストオーバーレイを追加できます。

![App Icon](PhotoFrame.png)
![App Screenshot](AppScreenShot_JP.png)

## 📋 主な機能

- **一括書き出し**: 複数の写真や動画を一度に同じ設定でフレーム書き出しできます。
- **柔軟なフレームレイアウト**: 標準的な比率（1:1、4:5、3:2、16:9 など）のほか、カスタム比率も指定可能です。
- **精密な位置調整**:
    - スライダーで写真の**垂直・水平位置**をフレーム内で細かく調整できます。
    - フレーム幅は `0` から設定でき、ボーダーレスに近い見た目から広い余白まで作れます。
- **写真枠**: 写真そのものと同じ比率で乗る追加の枠を、色と幅を指定してオプション表示できます。
- **LUT対応**: グループごとに `.cube` LUT を割り当て、適用率を調整しながら、写真と動画の両方へ同じ色変換を適用できます。
- **Text Layers**:
    - 複数のテキストレイヤーを追加できます。
    - 各レイヤーで `{Camera}`、`{Lens}`、`{Focal}`、`{FStop}`、`{Shutter}`、`{ISO}`、`{Date}`、`{Year}`、`{Month}`、`{Day}` などのEXIFタグを使えます。
    - さらに、現在の写真にあるタグ名なら `{TagName}` 形式でそのまま入力できます。
    - フォント、色、サイズ、X/Y位置、揃え、表示・非表示をレイヤーごとに設定できます。
- **写真グループ**: 左パネルで写真をグループ分けし、マウスで並び替えながら、グループ単位で同じレイアウト設定を適用できます。
- **Groupごとのスライドショー設定**: Groupごとに、表示秒数、動画の長さの扱い、追加音声、元動画音声のミックス、フェードイン / フェードアウトを保持できます。
- **スライドショープレビューと書き出し**: プレビューを `Photo` と `Slideshow` で切り替え、タイミングと音声バランスをその場で調整しながら、現在の Group を MOV のスライドショーとして書き出せます。
- **全画面スライドショー**: スライドショープレビューを全画面再生でき、必要に応じて再生後に次の Group へ自動で進められます。
- **Item欄サイズ調整**: 左パネルのスライダーで写真行の高さを詰めたり広げたりできます。
- **ワークスペース復元**: 次回起動時に、前回の Group 構成、設定、写真の所属を復元できます。
- **ワークスペース保護操作**: ワークスペース復旧に失敗した場合は自動保存を停止し、警告バナーから保存再開またはワークスペース初期化を選べます。
- **高速なインタラクティブプレビュー**: キャッシュ済みプレビューとライブテキスト描画により、テキスト編集や位置調整が軽快です。
- **動画プレビューと書き出し**: MOV / MP4 をフレーム内でプレビューし、そのまま全フレームへ枠とテキストを焼き込んだ MOV として書き出せます。
- **プリセット管理**: プリセットは A-Z 順で表示され、マウスオーバーでプレビュー、クリックで適用できます。保存、上書き、名前変更、削除、読み込み、書き出し、全消去にも対応しています。
- **柔軟な選択**: macOS標準（Finderなど）と同じ操作感で、Shift+クリックによる範囲選択やCmd+クリックによる個別選択に加え、`↑` / `↓` キーでの選択移動も可能です。
- **Preferences**: `PhotoFrame > Settings...` から、UI言語、カラーモード（`Midnight`、`Graphite`、`Black`、`Paper`、`Forest`）、フォント選択表示を切り替えられます。
- **複数ウインドウ対応**: 別の写真セットを複数ウインドウで並行して作業できます。

## 🚀 はじめに

### システム要件

- macOS 14.0以上
- Xcode Command Line Tools (`swift build`に必要)
- `exiftool`（任意。動画のメーカー固有メタデータをより多く読み取る場合）

### ビルドとインストール

付属のビルドスクリプトを実行して、アプリケーションのコンパイルとリソース（アイコンなど）のバンドルを行います。

```bash
bash build_app.sh
```

完了すると、プロジェクトのルートディレクトリに `PhotoFrame.app` が作成されます。

動画の `Lens`、`ISO`、`Focal`、`FStop`、`Shutter` などのメタデータをより多く表示したい場合は、あわせて `exiftool` をインストールしてください。`exiftool` が利用可能な環境では、標準の QuickTime/MP4 メタデータに加えて、カメラメーカー固有の動画メタデータも補助的に読み取ります。`exiftool` がない場合でもアプリは動作しますが、動画では表示できるタグが少なくなることがあります。

## 📖 操作マニュアル

### 1. 写真の追加
- **ドラッグ＆ドロップ**: FinderからJPEG / MOV / MP4 ファイルを左側のパネルにドラッグして追加できます。
- **ブラウズ**: 「+」ボタンをクリックするか、空のエリアをクリックしてファイル選択ダイアログから追加できます。

### 2. 選択とプレビュー
- **グループ**: 左パネルのフォルダ行で写真をグループ分けできます。プレビューと書き出しでは、各写真に所属グループの設定が使われます。
- **グループ並び替え**: グループ行を別のグループ行へドラッグすると、マウス操作でグループ順を入れ替えられます。
- **上部操作**: 左パネル上部の操作では、写真追加とグループ追加ができます。
- **グループ行メニュー**: 各グループ行の `...` メニューから、グループ名変更とグループ削除ができます。
- **Itemサイズ**: 左パネルの `Item Size` スライダーで、一覧行をコンパクトにもゆったりにも調整できます。
- **グループ移動**: 1枚または複数枚の写真を選択して、別のグループへドラッグして移動できます。
- **グループへ直接追加**: JPEG / MOV / MP4 ファイルをグループ行へ直接ドロップすると、そのグループに追加できます。
- **写真をクリア / Undo Clear**: Group を残したまま写真だけ消せます。直前のクリアは **Undo Clear** で戻せます。
- **複数選択の削除**: 複数選択した写真は、選択中のどれか1つの写真行にある `×` ボタンを押すとまとめて削除できます。
- **複数選択**: macOS標準の操作（**クリック**で単一選択、**Cmd+クリック**で追加/非選択の切り替え、**Shift+クリック**で範囲選択）が可能です。
- **キーボード移動**: 写真リストを選択したあと、`↑` / `↓` キーで現在の選択を前後へ移動できます。
- **プレビュー**: 選択されている項目のうち、最初の1件がプレビューパネルに表示されます。動画は枠付きの状態でその場で再生されます。
- **スライドショーモード**: プレビュー上部の切り替えで `Photo` と `Slideshow` を選べます。`Slideshow` では現在の Group 内の並び順をそのままプレビューし、MOV 書き出しにも同じ順序が使われます。
- **スライドショー再生操作**: プレビュー上部の再生 / 停止ボタン、またはプレビューのダブルクリックで再生 / 停止を切り替えられます。
- **フレーム設定の調整**: スライドショーではフレーム設定をライブ反映できないため、フレーム調整は `Photo` タブで行ってから `Slideshow` タブへ戻します。
- **写真順の並び替え**: Group 内で写真行をドラッグして順番を変えたり、別の Group へドラッグして移動できます。
- 設定を変更すると即座にプレビューに反映されるため、書き出し前に仕上がりを確認できます。
- **プレビュー品質**: 設定パネルの品質切り替えで、速度とディテールのバランスを調整できます。
- **複数ウインドウ**: **File > New Window** または `Cmd+N` で別ウインドウを開けます。

### 3. フレームとテキストの設定
- **グループ設定**: 設定パネルは、現在選択しているグループの設定を編集します。写真ごとに別レイアウトが必要な場合はグループを分けて使えます。
- **スライドショー設定**: スライドショープレビュー中は、コンパクトな設定行で動画の長さの扱い、1項目あたりの秒数、追加音声の選択 / 解除、追加音声の大きさ、元動画音声の使用と大きさ、フェードイン / フェードアウトを Group ごとに調整できます。
- **アスペクト比**: 外側のフレーム比率を選択します。「Custom」を選んだ場合は任意の比率（例: `4:5`）を入力できます。
- **写真の位置**: 「Photo Position」セクションのスライダーで、フレーム内の写真位置を垂直・水平方向に微調整できます。
- **フレーム幅**: 写真まわりの余白量を調整できます。`0` にすると余白なしに近いレイアウトになります。
- **写真枠**: 「Frame Style」セクションの `Show Photo Border` をオンにすると、写真と同じ比率の追加枠を表示できます。色と幅も変更できます。
- **LUT**: `LUT` セクションで `.cube` ファイルを選ぶと、現在のグループに属する写真と動画のプレビューおよび書き出しへ同じLUTを適用できます。
- **LUT適用率**: スライダーで `0%` から `100%` まで調整し、元画像とLUT適用後の見た目をブレンドできます。
- **Text Layers**:
    - 「Text Layers」セクションでレイヤーを追加できます。
    - `{Camera} • {Lens}` のようなテンプレートを入力できます。
    - 各レイヤーごとにフォント、色、文字サイズ、X/Y位置、揃えを設定できます。
    - 目のアイコンで表示切り替え、不要なら削除も可能です。
- **使用できるタグ**: `{Camera}`、`{Lens}`、`{Focal}`、`{FStop}`、`{Shutter}`、`{ISO}`、`{Date}`、`{Year}`、`{Month}`、`{Day}`。
- **現在の写真のタグ一覧**: Text Layers セクションで、プレビュー中の写真に実際に入っている metadata タグ名を確認できます。
- **動画メタデータについて**: 動画はファイルによって入っているタグ量の差が大きく、標準の QuickTime/MP4 タグしか持たないものもあります。`exiftool` が使える環境では、対応カメラの動画から `LensModel`、`LensID`、`WhiteBalance`、`FocusMode`、`FocusDistance`、`AFAreaMode`、`PrimaryAFPoint`、`PictureControlName`、`ShutterCount` などの追加タグも表示できる場合があります。

### 4. プリセットとPreferences
- **プリセット一覧**: 保存済みプリセットはプリセットポップオーバー内で A-Z 順に表示されます。
- **プリセットプレビュー**: プリセット名にマウスオーバーすると一時プレビューされ、クリックすると適用されます。
- **プリセット保存**: 設定パネル上部のプリセットメニューから現在のレイアウトを保存できます。
- **プリセットの読み込み / 書き出し / 貼り付け**: 読みやすい JSON ファイルとして共有し、他の Mac で読み込むか、preset 文字列を直接貼り付けて取り込めます。
- **プリセット上書き**: 各プリセット行の `...` メニューから、現在の設定で上書きできます。
- **プリセット名変更**: 各プリセット行の `...` メニューから **Rename Preset...** を選ぶと名前を変更できます。
- **プリセット削除**: 個別削除と全消去の両方に対応しています。
- **Group設定の書き出し / 読み込み**: **File** メニューから全 Group 設定をまとめて書き出し / 読み込みできます。
- **Preferences**: **PhotoFrame > Settings...** から以下を変更できます。
    - 表示言語（英語 / 日本語）
    - カラーモード（`Midnight`、`Graphite`、`Black`、`Paper`、`Forest`）
    - フォント選択表示（`Search` / `Full List`）
    - 音声ファイル用フォルダとLUT用フォルダのデフォルトパス
    - 全画面スライドショー再生後に次の Group へ自動で進むかどうか

### 5. 書き出し
- **Export Sel (...)**: リストで選択されている写真のみを書き出します（ボタン内に現在の選択枚数が表示されます）。
- **Export All**: リストにあるすべての写真を、選択状態に関わらず一括書き出しします。
- **書き出し設定**: 保存先フォルダを選ぶ前に、形式、画像サイズ、`Long Edge Custom`、JPEG品質、ファイル名プレフィックス、EXIF / メタデータ引き継ぎを設定できます。
- **動画書き出し**: 動画は `.mov` で書き出されます。画像サイズ設定は動画にも適用され、元動画に音声があれば保持されます。
- **スライドショー書き出し**: スライドショーモードでは Group 単位で書き出します。`Current Group` は現在の Group を 1 本の MOV にまとめ、`All Groups` は各 Group ごとに 1 本ずつ MOV を出力します。
- 保存先のディレクトリを選択すると、選択した形式とファイル名プレフィックスで保存されます。

## 🛡️ セキュリティと配布について

このアプリケーションは、Apple Developer証明書による署名が行われていない独立したツールです。別のMacで初めて実行する場合、Gatekeeperによる警告が表示されることがあります。

### 起動方法
1. Finderで `PhotoFrame.app` を表示します。
2. アプリを**右クリック（またはControl+クリック）**して、**「開く」**を選択します。
3. 確認ダイアログが表示されたら、再度**「開く」**をクリックします。

もし「壊れているため開けません」といった警告が表示される場合は、ターミナルで以下のコマンドを実行して隔離属性を削除してください。
```bash
xattr -cr PhotoFrame.app
```
